Pure CSS3 fly-out menu with responsive fallback, can be compltely customised to any style using the free online editor at http://swmenu.com

== Description ==
Visit http://swmenu.com and use the free online editor to make your own customised version of this pluggin.

* Pure simple CSS3 design
* No javascript used for most configurations
* Responsive fallback menu options for mobile devices
* PIE.htc integration for older Internet Explorer versions
* Vertical or Horizontal orientations
* Multilevel backgrounds using colors, gradients and images
* Submenu indicators for normal and hover conditions
* Special effects including text shadows, box shadows, slides and fades
* Variety of corner options and combinations
* Google webfonts integration

== Installation ==

1. Upload the pluggin to your wordpres site
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the pluggin in a position using the 'Widgets" menu in Wordpress
1. Assign the pluggin a menu to display it's items
